## üéØ Overview

This bootcamp provides hands-on training in Python's Pandas library, covering everything from basic data structures to advanced data manipulation techniques. Each module includes detailed Jupyter notebooks with examples, exercises, and real-world datasets.

**What you'll learn:**
- Pandas Series and DataFrame fundamentals
- Data cleaning and handling missing values
- Grouping and aggregation operations
- Merging, joining, and concatenating datasets
- Data input/output operations
- Advanced indexing and data manipulation

## üìñ Course Structure

### Module 1: Introduction to Pandas
**File:** `01-Introduction-to-Pandas.ipynb`
- Installing and importing Pandas
- Understanding What pandas is , and why use Pandas

### Module 2: Series Deep Dive
**File:** `02-Series.ipynb`
- Creating Series from various data sources
- Series attributes and methods
- Indexing and slicing operations
- Series operations and broadcasting
- Handling duplicate indices

### Module 3: DataFrames
**File:** `03-DataFrames.ipynb`
- Creating DataFrames from dictionaries, lists, and arrays
- Column and row operations
- Adding, removing, and renaming columns
- DataFrame indexing and selection
- Boolean filtering and conditional selection

### Module 4: Missing Data
**File:** `04-Missing-Data.ipynb`
- Identifying missing values (NaN, None, NaT)
- Dropping missing data with `dropna()`
- Filling missing values with `fillna()`
- Interpolation techniques
- Forward fill and backward fill methods

### Module 5: GroupBy Operations
**File:** `05-Groupby.ipynb`
- Understanding the split-apply-combine pattern
- Grouping by single and multiple columns
- Aggregation functions (sum, mean, count, etc.)
- Custom aggregation with `agg()` and `apply()`
- Transformation and filtering groups

### Module 6: Merging, Joining, and Concatenating
**File:** `06-Merging-Joining-and-Concatenating.ipynb`
- Concatenating DataFrames with `concat()`
- Merging DataFrames with `merge()`
- Understanding join types (inner, outer, left, right)
- Joining on indices with `join()`
- Handling overlapping column names

### Module 7: Operations
**File:** `07-Operations.ipynb`
- Arithmetic operations on DataFrames
- Statistical operations and descriptive statistics
- Sorting by values and index
- Ranking data
- Applying custom functions with `apply()`, `map()`, and `applymap()`

### Module 8: Data Input and Output
**File:** `08-Data-Input-and-Output.ipynb`
- Reading and writing CSV files
- Working with Excel files (`.xlsx`, `.xls`)
- JSON data handling
- SQL database connections
- HTML tables and web scraping basics
- Pickle files for Python objects
